# MVP 范围与实现优先级

## MVP 目标

第一版建议控制在 30 到 45 分钟一局。

MVP 的目标不是做完整长篇，而是验证核心循环：

> 逃避会让当下变简单，但会污染未来；面对会让当下更痛苦，但能把痛苦变成证据。

只要 MVP 能让玩家清楚感受到这条循环，就可以暂时压缩章节数量、角色数量和结局数量。

## MVP 内容范围

最小可玩内容：

- 1 名固定主角：林澈
- 4 个站点：上车、遗失物招领、医院站、清算站
- 3 名关键乘客：沈眠、许向东、周姐
- 50 到 70 张卡牌
- 12 到 18 个站点事件
- 3 个结局：循环、公开、清算失败

## MVP 站点规划

### 站点 1：上车

功能目标：

- 教学基础抽牌、出牌、行动力和事件目标。
- 引入 0:13 末班车规则。
- 引入许向东和“你见过我女儿吗？”的问题。

推荐事件：

- 空车厢
- 无脸检票员
- 抱花老人

主要产出：

- 旧车票
- 许向东关系变化
- 第一批普通负面牌

### 站点 2：遗失物招领

功能目标：

- 引入证据牌和物品牌。
- 让玩家第一次发现沈眠的视频。
- 强化“调查会增加压力，但能获得证据”的体验。

推荐事件：

- 破碎手机
- 堆满书包的站台
- 无人接听的电话
- 被水泡坏的工牌

主要产出：

- 破碎手机
- 沈眠的视频
- 未接来电
- 最后通话

### 站点 3：医院站

功能目标：

- 引入林澈与事故报道的关系。
- 引入周姐，建立“乘客关系会影响清算”的预期。
- 让玩家看到面对记忆会扣理智，但能获得关键证据。

推荐事件：

- 湿透的病历
- 记者证
- 擦不掉的血迹
- 改过的口供

主要产出：

- 记者证
- 清洁工口供
- 周姐支援牌
- 记忆断片

### 站点 4：清算站

功能目标：

- 根据整局表现进行结算。
- 让证据、乘客关系、负面牌和关键选择产生集中反馈。
- 分流到 MVP 的 3 个结局。

推荐事件：

- 检票
- 交出不愿承认的东西
- 乘客证词
- 最后一班列车

主要判定项：

- 证据数量
- 是否拥有沈眠的视频
- 是否拥有清洁工口供
- 许向东关系
- 周姐关系
- 严重负面牌数量
- 崩溃次数

## MVP 结局

### 循环结局

触发条件建议：

- 证据不足。
- 关键乘客关系过低。
- 严重负面牌过多。
- 清算站检票失败。

结局表现：

林澈回到现实后以为一切结束。第二天凌晨，他再次站在 0:13 的站台上。

### 公开结局

触发条件建议：

- 拥有沈眠的视频。
- 至少拥有 2 个关键证据。
- 至少 1 名关键乘客愿意作证。
- 清算站通过检票。

结局表现：

林澈带着录音和视频回到现实，发布迟到三年的报道。真相引发调查，死者不会复活，但许向东终于想起女儿已经不在。

### 清算失败结局

触发条件建议：

- 玩家抵达清算站，但负面牌、崩溃次数或推责选择过多。
- 玩家有部分证据，但没有承认自己的责任。

结局表现：

检票员判定林澈“无票乘车”。列车继续行驶，林澈被留在车上，成为下一轮循环的一部分。

## 首版系统优先级

### 必须实现

- 牌库、手牌、弃牌堆、抽牌、洗牌。
- 行动力和出牌费用。
- 事件目标值与回合倒计时。
- 理智、压力、证据、乘客关系。
- 普通负面牌加入牌库。
- 严重负面牌和崩溃计数。
- 站点推进。
- 清算站结局判定。

### 可以简化

- 分支线路可以先不做，只做线性 4 站。
- 角色立绘和叙事插图可以先用占位图。
- 卡牌稀有度可以先不做。
- 构筑奖励可以先固定，不做复杂随机池。
- 隐藏真结局可以暂不进入 MVP。

### 暂不建议实现

- 多主角。
- 完整 7 章长线。
- 大量随机地图。
- 复杂遗物系统。
- 多周目永久成长。
- 完整语音和演出系统。

## 推荐数据结构方向

### Card

卡牌建议至少包含：

- id
- name
- type
- cost
- description
- tags
- effects
- relatedCharacter
- relatedEvidence

### Event

事件建议至少包含：

- id
- name
- stationId
- objectiveName
- objectiveValue
- turnLimit
- failurePenalty
- successReward
- specialSolutions
- narrativeText

### Station

站点建议至少包含：

- id
- name
- description
- eventPool
- requiredEvents
- rewards
- nextStationIds

### RunState

单局状态建议至少包含：

- currentStation
- sanity
- pressure
- collapseCount
- evidence
- passengerRelations
- deck
- hand
- discardPile
- negativeCards
- completedEvents
- majorChoices

## 第一批内容建议

### 初始牌组方向

林澈的初始牌组应该体现“记者能力”和“逃避倾向”并存：

- 观察
- 采访
- 记录
- 追问
- 解释
- 转移话题
- 深呼吸
- 撒谎
- 暂缓
- 删除草稿

### 负面牌方向

普通负面牌：

- 焦虑
- 愧疚
- 幻听
- 逃避
- 走神

严重负面牌：

- 自我否认
- 记忆断片
- 未接来电
- 恐慌发作

### 证据牌方向

- 沈眠的视频
- 最后通话
- 清洁工口供
- 记者证
- 施工报警记录

## 验证标准

MVP 完成后，应重点验证以下问题：

- 玩家是否理解“逃避短期有利、长期有害”。
- 玩家是否愿意为了证据承受压力和理智损失。
- 负面牌污染是否能被明显感知。
- 清算站是否让玩家觉得前面的选择被看见了。
- 三个结局是否能体现不同行为路径。
- 一局 30 到 45 分钟是否成立。
