# 核心机制与卡牌系统

## 设计目标

核心系统要服务于一个体验：玩家每次为了眼前收益而逃避，都会让后续牌库更差；玩家每次选择面对，都会承受短期痛苦，但可能获得证据、乘客支援和更好的结局条件。

系统关键词：

- 牌库污染
- 证据收集
- 压力管理
- 理智崩溃
- 乘客关系
- 限时事件
- 清算判定

## 推荐界面结构

游戏不采用自由移动，也不是传统双方站场对打。推荐界面是 16:9 卡牌桌面：

- 顶部：线路图，显示当前站点和可能分支。
- 中央：当前车厢或站点事件，包含事件目标、倒计时和叙事插图。
- 底部：玩家手牌，每回合抽 5 张。
- 左侧：当前乘客和关系状态。
- 右侧：理智、压力、证据、牌库、弃牌堆、负面牌数量。

## 核心玩法循环

1. 进入站点。
2. 从站点事件牌堆翻出 3 个事件。
3. 玩家选择事件处理顺序。
4. 每个事件以卡牌回合解决。
5. 玩家通过出牌推进线索、降低危险、管理压力或逃避惩罚。
6. 事件结算后获得证据牌、乘客牌、记忆牌或负面牌。
7. 抵达下一站。
8. 在清算站根据整局选择进行审判。

## 核心资源

### 理智

理智相当于生命值。恐怖事件、检票失败、关键记忆冲击会扣理智。

理智归 0 时触发一次崩溃。崩溃不会立刻结束游戏，但会强制抽取一张严重负面牌。一局内崩溃 3 次则失败。

设计作用：

- 约束玩家不能无限硬吃事件伤害。
- 让“面对真相”有真实代价。
- 通过崩溃机制把失败转化为牌库变化，而不是立刻结束游戏。

### 压力

压力是污染值。压力越高，负面牌越容易进入牌库。

压力达到上限时触发失控事件。高压力洗牌时可能加入“焦虑”“幻听”“逃避”等普通负面牌。

设计作用：

- 惩罚连续逃避或高风险操作。
- 形成延迟后果，让玩家感到未来正在变差。
- 与洗牌、抽牌、负面牌数量形成 roguelike 卡牌节奏。

### 证据

证据决定剧情深度和结局质量。证据不足也可能通关，但通常只能进入循环、逃票或不完整结局。

关键证据包括：

- 沈眠的视频
- 施工报警记录
- 被删采访录音
- 清洁工口供
- 事故前通话记录

设计作用：

- 作为主线推进条件。
- 作为清算站判定条件。
- 解锁隐藏事件、隐藏选项和更好结局。

### 乘客关系

帮助乘客可以获得支援牌和清算站证词。欺骗、放弃或利用乘客会降低关系，并可能在后期变成指控。

设计作用：

- 将角色互动转化为长期资源。
- 让玩家在事件中权衡“拿证据”和“帮人”。
- 在清算站把前期行为集中反馈给玩家。

### 时间

每个事件都有回合限制。时间耗尽会导致事件失败、扣理智、加入负面牌或错过证据。

设计作用：

- 防止玩家无限拖回合。
- 让快速逃避和慢速调查产生真实差异。
- 让事件有压迫感。

## 失败方式

### 理智崩溃失败

一局内崩溃 3 次，主角失去自我，被困在车上。

### 压力失控失败

压力多次爆表后，牌库被负面牌塞满，玩家无法有效处理事件，最终被列车吞没。

### 时间耗尽失败

如果玩家拖延过多，列车会直接抵达终点，但证据不足，只能进入坏结局。

### 清算失败

第六站清算时，如果证据太少、乘客关系太差、负面牌过多或关键记忆没有面对，检票员判定玩家“无票乘车”，进入循环结局。

## 卡牌类型

### 行动牌

行动牌代表玩家当下能做的事。

示例：

- 观察：抽 1 张牌。
- 采访：推进线索，但增加压力。
- 解释：降低怀疑。
- 深呼吸：回复理智或降低压力。
- 撒谎：大幅推进事件，但加入“愧疚”。

### 物品牌

物品牌通常来自剧情，是解锁隐藏选项的钥匙。

示例：

- 旧车票
- 破碎手机
- 白花
- 记者证
- 湿透的病历

### 证据牌

证据牌决定主线推进和结局。

示例：

- 沈眠的视频
- 最后通话
- 施工报警记录
- 被删采访录音

### 乘客牌

乘客牌代表乘客的帮助、信任或牵连。

示例：

- 抱花老人：抵消一次恐惧。
- 清洁工周姐：揭开一张隐藏事件牌。
- 沈眠的信任：复制一张证据牌的效果。

### 普通负面牌

普通负面牌用于小规模污染牌库。

示例：

- 焦虑：抽到时压力 +1。
- 愧疚：本回合少 1 行动力。
- 幻听：随机弃 1 张牌。

### 严重负面牌

严重负面牌通常在崩溃时获得，能临时改变规则。

示例：

- 自我否认：面对类牌 +1 费，逃避类牌 -1 费，持续到事件结束。
- 记忆断片：封锁一张证据牌，本事件不能使用。
- 未接来电：3 回合倒计时。如果不处理则再次崩溃；如果处理成功，转化为证据牌“最后通话”。
- 恐慌发作：本回合失去 2 点行动力，压力 +2。

## 事件模型

事件建议统一为以下结构，方便实现和内容扩展：

- 事件名称
- 当前站点
- 事件目标
- 目标数值
- 剩余回合
- 失败惩罚
- 可用特殊解法
- 成功奖励
- 叙事文本

### 示例事件：无脸检票员

目标：

- 怀疑值：8
- 剩余时间：3 回合
- 失败惩罚：扣 2 理智，加入“逃票记录”

玩家手牌示例：

- 解释：消除 2 点怀疑，消耗 1 行动力。
- 旧车票：如果拥有事故线索，消除 4 点怀疑，消耗 2 行动力。
- 撒谎：消除 5 点怀疑，但压力 +3，并加入“愧疚”。
- 深呼吸：回复 1 理智，消耗 1 行动力。
- 观察：抽 1 张牌，消耗 0 行动力。

核心选择：现在用撒谎快速通过，还是用证据和观察慢慢解决。

### 示例事件：无人接听的电话

目标：

- 接通信号：5
- 剩余时间：3 回合
- 失败惩罚：加入“未接来电”

特殊解法：

- 如果玩家使用“破碎手机”和“沈眠的视频”，可以接通事故前的最后通话，并获得证据牌“最后通话”。
